## Bank/ABD balance

**Bank balance gevonden:** [x] ja

**Bestand/functie**
- Veld: `ACNL_Player::BankAmount` (@0x6C2C, type `EncVal`).
- Lezen: `Game::DecryptValue(&player->BankAmount)` → `int`.
- Schrijven: `Game::EncryptValue(&player->BankAmount, value)`.
- Referentievoorbeeld: `bank()` in `src/features/MoneyCodes.cpp`:
  ```cpp
  ACNL_Player *player = Player::GetSaveData();
  if(!player) { /* melding */ return; }
  Keyboard kb(...); kb.SetSlider(0, 999999999, 10000);
  if(kb.Open(money) >= 0)
      Game::EncryptValue(&player->BankAmount, money);
  ```

**Adres/offset**
- Niet als los regio-adres nodig: `BankAmount` is een **structveld** binnen `ACNL_Player`
  (offset 0x6C2C binnen de player-save), bereikbaar via `Player::GetSaveData()`. Regio-onafhankelijk.
- Alleen de encrypt/decrypt-**functies** zijn regio-adressen:
  `EncryptValue` → `0x3036A4`, `DecryptValue` → `0x3037DC` (via `Address`-laag).

**Datatype**
- Opslag: [x] `EncVal` = **u64, versleuteld** ("secure value").
- Logische waarde: 32-bit `int` (encrypt neemt `int`, decrypt geeft `int`).

**Maximale veilige waarde**
- **999.999.999 bells** (ACNL bankmaximum; bevestigd door de slider-cap in `bank()`).
  Past ruim in u32/int.

**Per speler:** [x] ja — `BankAmount` zit in `ACNL_Player` (per spelerslot).
Verwant: `PocketMoney` (@0x6FA8, wallet), `DebtAmount` (huislening), `MeowCoupons`, `MedalAmount`.

**Checksum/secure value nodig:** [x] ja
- Direct als getal schrijven corrumpeert; **altijd** via `EncryptValue`/`DecryptValue`. De helper
  bestaat al, dus dit is opgelost (in tegenstelling tot wat het researchplan als open vraag had).

**Naar RAM of save-buffer?**
- `GetSaveData()` geeft de **live save-struct in RAM** (de "garden_plus" save in geheugen). Wijzigingen
  worden door de game bij de eerstvolgende save naar de SD weggeschreven. → reboot-persistentie
  verloopt via normale game-save (te bevestigen in praktijktest).

**Bestaande helperfunctie**
```cpp
// lezen
int current = Game::DecryptValue(&player->BankAmount);
// schrijven
Game::EncryptValue(&player->BankAmount, newValue);
```

## Praktijktest — OPEN (vereist 3DS/CFW)
```text

```
