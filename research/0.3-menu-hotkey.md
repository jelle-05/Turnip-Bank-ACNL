# Stap 0.3 — Menu en hotkey-systeem (resultaat)

> Bron: code-research (`main.cpp`, `MenuCreate.cpp`, `Plugin_Patches.cpp`).

## Menu en hotkey

**Plugin/menu titel wordt ingesteld in**
- Bestand: `src/main.cpp`
- Regel/functie: `main()` regel 103:
  ```cpp
  PluginMenu *menu = new PluginMenu(
      Color::White << "ACNL Vapecord Plugin " << region, majorV, minorV, revisV, NOTE);
  ```
- → Titel is **vrij aanpasbaar** (string + versie + `NOTE`). Voor een eigen plugin gewoon vervangen door
  bv. `"ACNL Turnip Bank " << region`.

**Menu-open hotkey wordt ingesteld in**
- **Niet hardcoded in deze repo.** Er staat nergens een expliciete "open menu"-toetscombinatie; het project
  zet in `PatchProcess()` (`Plugin_Patches.cpp`) alleen `ThreadPriority`, `CachedDrawMode`, HID-geheugen —
  geen menu-hotkey. De menu-open-combinatie komt dus uit **CTRPluginFramework zelf** (de framework-default,
  instelbaar in het CTRPF-overlay/settings-scherm).
- Individuele *features* hebben wél eigen hotkeys, via `EntryWithHotkey(...)` / `entry->Hotkeys[i]` /
  `Controller::IsKeysPressed(Key::...)` (zie `MenuCreate.cpp`, `AnimationCodes.cpp`, `DevCodes.cpp`).

**Hotkey is**
- [x] configureerbaar via CTRPF (framework-default menu-open combo; per-feature hotkeys instelbaar)
- ( ) hardcoded — alleen de per-feature voorbeelden, niet de menu-open zelf.

**Kan eigen plugin/menu**
- [x] ja. `InitMenu(menu)` (in `MenuCreate.cpp`) bouwt alle `MenuFolder`s/entries op. Voor een eigen
  Turnip-Bank-plugin kunnen we:
  1. de menu-titel in `main.cpp` wijzigen, en
  2. in `InitMenu` alleen onze eigen folder/entries appenden (gestripte build), of een nieuwe folder
     toevoegen aan de bestaande plugin.

**Branding / Vapecord-specifieke entries**
- De naam "Vapecord" zit in: de menu-titelstring (`main.cpp`), `TARGET := Vapecord_Public` (Makefile),
  de `.plgInfo`, SD-paden `E:/Vapecord/...` (`Files.h`), en localization-strings. Voor een echte eigen
  plugin moeten die punten meegenomen worden (titel, target-naam, plgInfo, opslagpaden).
- Anti-scam scherm (`ShowAntiScamScreen()`) en welcome message zijn los toggle-/verwijderbaar.

## Acceptatiecriterium — afgevinkt
- [x] Eigen plugin met eigen menu(-titel) is haalbaar; menu-open hotkey is een CTRPF-instelling
      (niet hardcoded), per-feature hotkeys zijn beschikbaar via het bestaande `EntryWithHotkey`-mechanisme.

## Aandachtspunt voor later
- Of we een **volledig eigen plugin** maken of een **folder binnen Vapecord** is een productbeslissing
  (onderhoud, updates, branding). Technisch kan beide; een eigen gestripte build is schoner voor distributie.
